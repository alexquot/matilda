Getting started

To begin with you need to install Matilda.

npm install --save @alexquot/matilda
yarn add @alexquot/matilda

Then you'll be able to include it in your SASS code as simple as follows.

@import "@alexquot/matilda/scss/matilda";

Buttons

Common buttons

Basic buttons have the ripple effect. You can also use .ripple class and ripple($color: $black) mixin in your SASS code.

Outline buttons

Outline buttons are transparent and don't have shadows at all.

Sizes

Rounded buttons

As an addition to Boostrap 4 calsses you can use .btn-rounded class. It adapts button's size according to the previous .btn-sm or .btn-lg class usage.
