Usage

To begin with you need to install Matilda. It was released in npm under @alexquot/matilda package. If you want you can use jsDelivr CDN, go on right to the download page. You can also try Matilda and copy-paste the stylesheet into your before all other stylesheets.

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@alexquot/matilda/dist/matilda.css">

But if you want to customize it (e.g. change color palette) you need to install it and compile manually. Simply install it.

npm install --save @alexquot/matilda

And import in your SASS code.

@import "@alexquot/matilda/scss/matilda";

Talking about Bootstrap, there is no need for importing it as soon as it's included in Matilda build. You'll likely to want to use some JavaScript with it. It's ok to use Matilda with official Bootstrap JavaScript plugins, but of course you can use something like ngx-bootstrap or reactstrap.

Buttons

Buttons have the ripple effect. You can also use .ripple class and ripple($color: $black) mixin in your SASS code.

As an addition to Boostrap 4 calsses you can use .btn-rounded class. It adapts button's size according to the previous .btn-sm or .btn-lg class usage.

Typography

Basics

Roboto fonts are recommended, you should include them by yourself. Please, don't forget to include different font weights in your imports. It's recommended to make them look something like this.

<link href="https://fonts.googleapis.com/css?family=Roboto:300,400,500" rel="stylesheet">
<link href="https://fonts.googleapis.com/css?family=Roboto+Mono:300,400,500" rel="stylesheet">
