Matilda

Material Design theme for Bootstrap 4

About

Matilda is a Material Design theme for Bootstrap 4. The goals of this project are to provide a theme that looks similar to Material Guidelines, but not to overdose it with complexity. There's also won't be any additional JS plugins, so you can use Matilda flawlessly with any other framework like Angular or React. Currently it's in early stage development, but one can find it's quite usable already.
