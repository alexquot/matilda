About

Matilda is a kinda Material theme for Bootstrap 4. The goals of this project are to provide a theme that looks similar to Material Guidelines, but not to overdose it with complexity. There's a need for a simple but beautiful Bootstrap themes, I want to keep everything in a single CSS file. So, no JS plugins at all.

Currently it's in early stage development, but one can find it's quite usable already.

Helping the project

It would be nice to see any activity in project's GitHub page. Feel free to fill issues and create pull requests.

Buttons

Common buttons

Basic buttons have the ripple effect. You can also use .ripple class and ripple() mixin in your SASS code. Yes, there's also changed shadows, which were hardcoded by default by Bootstrap team members.

Outline buttons

Outline buttons are transparent and don't have shadows at all. They also don't have ripple effect, so basically they're flat.

Sizes

Fab buttons

As an addition to Boostrap 4 calsses you can use .btn-fab class now. It adapt button's size according to the previous .btn-{sm, lg} class usage.
